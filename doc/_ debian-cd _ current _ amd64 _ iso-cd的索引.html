这个目录里有什么？

这些文件包含Debian GNU / Linux操作系统的安装程序和其他软件。此目录中的文件专门用于amd64体系结构。

我该如何使用这些文件？

这里的文件是完整的ISO映像，随时可以使用。

下载完所需的所有ISO映像后，通常需要将它们写入安装介质。

我需要什么尺寸和类型的媒体？

这里描述的图像的大小至少可写入可写CD介质，但如果需要可以写入更大的介质。

为方便起见，这些图像也可以直接写入USB记忆棒。只要您的计算机直接从该USB记忆棒启动，就应该以这种方式启动Debian安装程序。

这里有很多文件！我需要所有这些吗？

在大多数情况下，没有必要下载和使用所有这些图像，以便能够在您的计算机上安装Debian。Debian附带了大量的软件包，因此为什么需要这么多磁盘来完成整套软件包。大多数典型用户只需要这些软件包的一小部分。

什么是netinst图像？

netinst这里的CD是一个小的CD映像，它只包含核心Debian安装程序代码和一小组核心文本模式程序（在Debian中称为“标准”）。要安装桌面或其他常用软件，您还需要Internet连接或其他一些Debian CD / DVD映像。

什么是mac netinst图像？

mac netinst此处的CD是netinst CD映像的特殊版本，专门用于较旧的64位Intel Macintosh计算机。它也可能适用于大多数其他amd64机器，但它不包含某些人需要的UEFI引导文件。有关更多信息，请参阅 Debian Wiki。

如何验证我的下载是否正确以及Debian创建的内容是什么？

这里有文件（SHA1SUMS，SHA256SUMS等），其中包含图像的校验和。这些校验和文件也已签名 - 请参阅SHA1SUMS.sign，SHA256SUMS.sign等。下载图像后，您可以检查：

  • 它的校验和与校验和文件中的校验和匹配; 和
  • 校验和文件没有被篡改。

有关如何执行这些步骤的详细信息，请阅读验证指南。

只有前几张图片可用！其余的在哪里？

我们不为所有体系结构存储/提供全套ISO映像，以减少镜像占用的空间量。您可以 使用jigdo工具重新创建缺少的ISO映像。

非免费固件

这是一个官方的 Debian图像构建，因此只包括自由软件。

为了方便某些用户，还有一个替代的非官方 netinst CD版本，其中包括非免费固件，以便为某些笨拙的硬件提供额外支持。 如果您需要该CD映像，请查看 / cdimage / unofficial / non-free / cd-including-firmware /。
